Detailed Description

Class DSTranscript

Each Transcript contains:

  • id: Integer - A unique transcript ID (Autoincremented).
  • transcript_id: String - The unique Transcript ID.
  • data_source: String - The name of the source.
  • gene_id: String - The unique gene ID.
  • strand: String - The strand of the transcript.
  • raw_start_pos: String - The position of the first nucleotide of the transcript (in the original annotation version).
  • raw_end_pos: String - The position of the last nucleotide of the transcript (in the original annotation version).
  • start_pos: String - The position of the first nucleotide of the transcript (in the current annotation version).
  • end_pos: String - The position of the last nucleotide of the transcript (in the current annotation version).
  • sequence: String - The sequence of nucleotides get using the genomic coordinates (in the current annotation version).
  • raw_cds_start_pos: String - The position of the start codon of the CDS (in the original annotation version).
  • raw_cds_stop_pos: String - The position of the stop codon of the CDS (in the original annotation version).
  • cds_start_pos: String - The position of the start codon of the CDS (in the current annotation version).
  • cds_stop_pos: String - The position of the stop codon of the CDS (in the current annotation version).
  • gencode_basic: Boolean - Is the gencode basic reported?
  • rna_biotype: String - The RNA biotype ('mRNA', 'lncRNA'...).
