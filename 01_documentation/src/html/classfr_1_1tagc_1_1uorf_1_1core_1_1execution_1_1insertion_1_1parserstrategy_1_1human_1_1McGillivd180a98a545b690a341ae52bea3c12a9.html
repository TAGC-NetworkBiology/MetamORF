Detailed Description

Source: McGillivray et al., 2018.

Description of the columns of the raw data file: NB: Columns 0 to 3 have been generated from the original ORF ID ("uORF_ID" column in the raw data file) ORF ID was formatted as: Ensembl transcript ID + "uORF_" + start sequence + "." + unique number [0]: TranscriptID: Ensembl transcript ID - String [1]: TranscriptVersion: Ensembl transcript version - Integer [2]: ORFstart: ORF start sequence (as "uORF_START") - String [3]: ORFnum: Unique number in ORF unique ID

McGillivray2018PPD
