Detailed Description

Class DSORFTranscriptAsso

Each ORFTranscriptAsso contains:

  • id: Integer - A unique ORF-transcript association ID (Autoincremented).
  • data_source: String - The name of the source.
  • transcript_id: String - The unique transcript ID.
  • uniq_orf_id: Integer - The ORF unique ID as defined in the ORF object.
  • predicted: Boolean - Was the ORF predicted by computational biology?
  • ribo_seq: Boolean - Was the ORF detected or predicted from Ribo-seq data?
  • cell_context: String - The cell line or context in which the ORF was described.
  • orf_id: String - The ORF ID as defined in the publication where it was described (cf. data_source).
  • raw_sequence: String - The nucleotidic sequence of the ORF (as provided in the dataset where it was initially reported).
  • raw_sequence_aa: String - The amino acid sequence of the ORF (as provided in the dataset where it was initially reported).
  • provided_category: String - The category provided in the original dataset.
  • start_codon_seq: String - The nucleotidic sequence of the start codon (as provided in the dataset).
  • ms_info: Boolean - Is there any mass spectrometry evidence about the ORF-encoded peptide?
  • kozak_context: String - The Kozak context of the start codon.
  • orf_length: Integer - The ORF legth as provided by the source (in nucleotides).
  • orf_score: Float - The ORF score (provided by the source).
  • phylocsf: Float - The PhyloCSF score of the ORF (provided).
  • phastcons: Float - The PhastCons score of the ORF (provided).
  • floss: Float - The FLOSS score of the ORF (provided).
  • floss_class: String - The FLOSS class of the ORF (provided).
  • gen_len_eq_orf_len: Boolean - Does the genomic length (computed using genomic coordinates) equal the provided ORF length?
