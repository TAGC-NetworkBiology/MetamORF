Detailed Description

GeneRefLogger

This class is a singleton which allows to log warnings relative to the gene cross references (gene IDs / symbol / aliases...) anywhere in the program.

/!\ There is one single logging level for this logger: WARNING. NB: Logs created with this logger are not displayed on the console.
