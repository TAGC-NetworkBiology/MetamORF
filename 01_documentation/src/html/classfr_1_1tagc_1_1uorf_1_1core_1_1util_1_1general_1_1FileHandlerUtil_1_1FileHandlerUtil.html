dict_to_csv

This is a static method that allows to save the content of a dictionary into a CSV file.

Parameters
output_folderString - The path to the output directory.
filenameString - The name of the file in which the dictionary has to be saved.
dictDictionary - The dictionary to save.
file_descString - A description of the file to log.
sortBoolean - Does the dictionary needs to be writer in the same order as sorted keys? False by default.
sepString - The field delimiter for the output file. Comma (',') by default.
extString - The extension of the file. 'csv' by default.
hdrList - An eventual list of strings that has to be used to create the headers.
key_funcFunction - An optional function to apply to the keys of the dictionary. This would allow for instance to join a list into a string with a particular separator. The function provided could be a lambda function. By default, the identity function.
val_funcFunction - An optional function to apply to the values of the dictionary. This would allow for instance to join a list into a string with a particular separator. The function provided could be a lambda function. By default, the identity function.
unlist_keyBool - If True, the keys have to be list (or tuples) that will be unlist such as each element will constitute the value of a column. False by default. NB: This parameter is applied AFTER the function.
unlist_valBool - If True, the values have to be list (or tuples) that will be unlist such as each element will constitute the value of a column. False by default. NB: This parameter is applied AFTER the function.
Exceptions
DenCellORFExceptionWhen the header provided does not contain exactly two items.
