Detailed Description

Class DSORF

Each DSORF contains:

  • id: Integer - Unique ORF ID (Autoincremented).
  • data_source: String - The name of the source.
  • chromosome: String - The chromosome name harboring the ORF.
  • raw_strand: String - The strand (+/-) in which the ORF is encoded (in the original genome version).
  • raw_start_pos: String - The position of the start codon as genomic coordinates (in the original genome version).
  • raw_stop_pos: String - The position of the stop codon as genomic coordinates (in the original genome version).
  • spliced: Boolean - Is the ORF spliced?
  • raw_splice_starts: String - The positions of the starts of the several "exonic" parts of the ORF (in the original genome version).
  • raw_splice_ends: String - The positions of the ends of the several "exonic" parts of the ORF (in the original genome version).
  • spliced_parts_count: Integer - The number of "exonic" parts in the ORF.
  • strand: String - The strand (+/-) in which the ORF is encoded (in the current genome version).
  • start_pos: String - The position of the start codon as genomic coordinates (in the current genome version).
  • stop_pos: String - The position of the stop codon as genomic coordinates (in the current genome version).
  • splice_starts: String - The positions of the starts of the several "exonic" parts of the ORF as genomic coordinates.
  • splice_ends: String - The positions of the ends of the several "exonic" parts of the ORF as genomic coordinates.
  • sequence: String - The sequence get using the genomic coordinates (in the current genome version).
  • sequence_aa: String - The sequence get by translated the nucleic sequence.
  • genomic_length: Integer - The ORF length, computed using genomic coordinates.
