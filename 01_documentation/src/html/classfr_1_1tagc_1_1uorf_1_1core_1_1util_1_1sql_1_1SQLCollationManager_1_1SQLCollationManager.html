Detailed Description

SQLCollationManager

This class is a singleton aiming to manage the collation of the database. The value of the collation is useful to define the column which aim to receive strings (such as VARCHAR or TEXT types).

set_db_collation

This method allows to set the appropriate collation according to the database type. If the database type is not provided, it will set the collation to the value of the default type. See the documentation of the SQLConstants file for more information.
