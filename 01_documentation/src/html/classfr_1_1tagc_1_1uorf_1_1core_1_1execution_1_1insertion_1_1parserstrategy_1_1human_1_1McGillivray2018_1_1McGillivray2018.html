Detailed Description

Source: McGillivray et al., 2018.

Description of the columns of the raw data file: [0]: uORF_ID: ORF ID - String ORF ID formatted as: Ensembl transcript ID + "uORF_" + start sequence + "." + unique number [1]: uORF_score: ORF score - Float [2]: start_codon: Start codon sequence - String [3]: chromosome: Chromosome name - String [4]: strand: Strand - String [5]: start_coordinate: Genomic coordinates of the start codon (coordinates of the 1st nucleotide) - Integer [6]: end_coordinate: Genomic coordinates of the stop codon (coordinates of the last nucleotide) - Integer [7]: gene: Gene symbol - String [8]: additional_transcripts: ORF ID containing the name of the alternative transcript (CSV) - String [9]: type: ORF classification - String [10]: peptide_score: Peptide score - Float

McGillivray2018

This class is a ParserStrategy allowing to parse data from McGillivray et al., 2018.
