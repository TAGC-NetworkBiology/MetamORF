Detailed Description

RefException

This class allows to raise exceptions related to problems about gene references anywhere in the program. It inherits from the DenCellORFException.

Constructor of RefException

Instance variable:

  • message: String - The message to log about the error.
  • exception: Exception - The exception caught.
  • gene: Gene - A Gene object.

The gene object registered as attribute of the exception may be either the updated gene (in the case of the RefGeneUpdateException class) or a gene that is related to the exception, such as a "fake" gene that has been created to respect the relational integrity of the database when one was not able to find an unique gene for a unique (symbol, chromosome) pair.
